# Vacation Budget Planner
 
## Project Description
 
Vacation Budget Planner is a web application that helps users plan and budget their dream vacations. The app allows users to create vacation plans, set budgets, specify transportation methods, and manage excursions available at their destination. Users can compare costs with and without excursions to make informed vacation planning decisions.
 
## Problem It Solves
 
Planning a vacation involves juggling multiple costs and options. This app centralizes budget planning, allowing users to:
- Set overall vacation budgets
- Track excursion options and prices
- Compare total costs with different excursion combinations
- Stay within budget while exploring their destination options
## Intended Users
 
- Travelers planning vacations
- Budget-conscious holidaymakers
- Groups planning trips together
- Anyone who wants to optimize their vacation spending
## Key Features & User Actions
 
- **User Management:** Register, login, edit profile, delete account
- **Vacation Management:** Create, view, edit, and delete vacation plans
- **Vacation Details:** Specify destination, travel dates, total budget, and transportation method
- **Excursion Management:** View available excursions for a vacation, add excursions to a trip, remove excursions
- **Budget Tracking:** Automatically calculate total trip cost (base budget + excursion costs)
- **Cost Comparison:** View trip cost with and without selected excursions
